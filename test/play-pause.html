Normalized play / pause and mute API tests

Tests each provider’s play/pause API: mount the embed, assert currentTime === 0 before play, then provider.play(), onPlay fired, await provider.paused, provider.pause(), onPause fired, provider.seek(seconds), and assert currentTime ≈ seek target. Also tests provider.mute(), provider.muted, provider.unmute(), and provider.volume / provider.setVolume(0–1) where supported.
