Normalized onBuffering callback tests

Tests that each provider calls the normalized onBuffering callback when playback buffers. The test mounts the embed with onBuffering, starts playback (and may seek to encourage buffering), then waits. If onBuffering is invoked at least once, the test passes. If no buffering occurs within the wait window, the result is "No buffering occurred (network-dependent)" and still passes, since buffering is not guaranteed.
