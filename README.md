# embed-kit

Normalized API across different embed sources (YouTube, Twitch, TikTok, and others).

## Setup

```bash
npm install
npm run build
```

## Scripts

- **`npm run build`** — Compile TypeScript to `dist/`
- **`npm run clean`** — Remove `dist/`

## Development

Source lives in `src/`. The package entry is `src/index.ts`.
